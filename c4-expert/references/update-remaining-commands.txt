Control4 Commands to Update - Step-by-Step Guide
================================================

ALL COMMANDS NEED:
- IP Address: homesk.kbl55.com (or 192.168.20.19)
- Port: 50261
- Encoding: ASCII

==========================================
DEVICE: Master Thermostat (Device ID: 783)
==========================================

1. Open device: Master Thermostat
2. Go to Programming tab → Event: 103
3. Find: "Send generic TCP command" action
4. Update Command field to:
{"type": "overRideSnowEvent", "loc":"Outside","data": "Override snow melt on for 6 hours"}

5. Update IP Address to: homesk.kbl55.com
6. Update Port to: 50261


===========================================
DEVICE: Driveway North (Device ID: 367)
===========================================

EVENT 1 (OFF):
1. Open device: Driveway North
2. Go to Programming tab → Event: 1
3. Find: "Send generic TCP command" action
4. Update Command field to:
{"type": "snowStop", "loc":"drivewayNorth","data": PARAM{781,1133}}

5. Update IP Address to: homesk.kbl55.com
6. Update Port to: 50261

EVENT 2 (ON):

2. Go to Programming tab → Event: 2
3. Find: "Send generic TCP command" action
